# Cyber Services Analytics Platform (CSAP)

Welcome to the **Cyber Services Analytics Platform (CSAP)**. CSAP is a professional analytics and reporting system specifically designed for cybersecurity services teams to analyze productivity, capacity, backlog, SLA compliance, and operational trends.

---

## Important Architectural Distinction

- **CSAP is NOT an ITSM system.**
- **CSAP does NOT replace Archer.**
- **CSAP does NOT manage tickets or operational workflows directly.**

CSAP serves strictly as an **analytical platform**. It ingests daily CSV export files generated by Archer, processes the tabular data, calculates advanced operational metrics, and generates interactive dashboards, executive reports, and AI-driven insights.

---

## 1. Project Philosophy

This repository is designed to grow and sustain itself for many years. We place immense emphasis on:
1. **Architecture over Speed**: Building modular, scalable structures that prevent technical debt.
2. **Legibility over Complexity**: Code must be clear, well-commented, and easily understandable.
3. **Maintainability over Short Code**: Writing explicit, well-typed code is preferred over "clever" one-liners.

## 2. Technology Stack

### Backend
- **Language**: Python 3.13
- **Framework**: FastAPI (for modular API design)
- **ORM & Migrations**: SQLAlchemy 2.x and Alembic
- **Validation**: Pydantic v2
- **Database & Cache**: PostgreSQL (primary store) and Redis (caching and rate-limiting)
- **Data Engineering**: Polars (used for high-performance processing of the daily CSV files)
- **Testing**: Pytest

### Frontend
- **Framework**: React with TypeScript and Vite
- **UI Library**: Mantine
- **Grid Components**: AG Grid Enterprise (for high-volume tabular data view/analysis)
- **Visualizations**: Apache ECharts
- **State & Queries**: TanStack Query and Zustand
- **Form Management**: React Hook Form with Zod validation
- **Testing**: Vitest (unit/component) and Playwright (end-to-end)

### Infrastructure
- **Containerization**: Docker & Docker Compose
- **Web Server / Reverse Proxy**: Nginx

### Quality & Linting
- **Python**: Ruff, MyPy
- **TypeScript/React**: ESLint, Prettier
