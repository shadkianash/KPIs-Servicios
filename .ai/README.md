# AI Agent Instructions & Context Root

Welcome, AI Agent! This directory (`.ai/`) acts as your persistent context and rules interface.

Before proposing, modifying, or writing any code, you are **required** to read and adhere to all specifications, domain definitions, and standards outlined in this folder. This ensures the project preserves its high-quality architectural baseline.

---

## Folder Content Guide

1. **[PROJECT.md](PROJECT.md)**: Details the goals, scope, and non-goals of the Cyber Services Analytics Platform (CSAP).
2. **[ARCHITECTURE.md](ARCHITECTURE.md)**: Defines core architectures, decoupled boundaries, and overall system design.
3. **[STACK.md](STACK.md)**: Enforces the exact technologies, frameworks, libraries, and quality tools (Ruff, MyPy, ESLint, Prettier, etc.) approved for use.
4. **[RULES.md](RULES.md)**: Outlines strict coding, typing, and formatting standards.
5. **[DOMAIN.md](DOMAIN.md)**: Maps out business domains, entities, glossaries, and calculation math rules.
6. **[CONSTITUTION.md](CONSTITUTION.md)**: Establishes behavioral tenets and core coding beliefs.
7. **[IMPLEMENTATION_PLAN.md](IMPLEMENTATION_PLAN.md)**: High-level long-term vision divided into systematic phases.
8. **[CURRENT_STATE.md](CURRENT_STATE.md)**: Tracks the active status, versioning, and readiness of repository folders.
9. **[NEXT_TASK.md](NEXT_TASK.md)**: The definitive, direct next step for execution.
10. **[REVIEW_CHECKLIST.md](REVIEW_CHECKLIST.md)**: A checklist of criteria to confirm your work meets standard before submitting PRs.
11. **[PROMPTS/README.md](PROMPTS/README.md)**: Prompt blueprints to generate consistent code additions.

---

## How to Proceed
1. Read **`PROJECT.md`** and **`ARCHITECTURE.md`** to grasp what you are modifying.
2. Read **`RULES.md`** and **`STACK.md`** to align with the technology ecosystem.
3. Check **`NEXT_TASK.md`** to verify what needs to be coded next.
4. When finished, consult **`REVIEW_CHECKLIST.md`** and update **`CURRENT_STATE.md`** and **`CHANGELOG.md`** before opening a Pull Request.
