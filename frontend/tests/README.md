# Frontend Test Workspace

This directory structures the complete testing matrix for the React web application.

## Subdirectories
- `unit/`: Component and hooks tests executing on Vitest and React Testing Library (simulating rendering and virtual DOM actions).
- `e2e/`: End-to-end integration flow scenarios executing on Playwright, verifying automated user logins, file ingestion forms uploads, and visual charts renderings in headless chromium/firefox/webkit.
