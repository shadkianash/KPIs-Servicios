# React Components

This folder contains modular, reusable UI components for the CSAP dashboard.

## Structure
- `/common`: Standard layouts, buttons, loaders, and navigation cards.
- `/charts`: Specialized components wrapping Apache ECharts (e.g. `SlaComplianceGauge`, `BacklogAgingChart`, `ResourceHeatmap`).
- `/tables`: Wrap AG Grid Enterprise setups to present dense operational databases.
- `/forms`: Standard form input structures tied to React Hook Form and validated with Zod.
