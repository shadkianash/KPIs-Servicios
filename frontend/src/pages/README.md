# Dashboard View Pages

This directory houses the root visual page layouts of our Single Page Application.

## Primary Views
- `DashboardPage.tsx`: Main landing, showcasing SLA gauges, backlog summaries, and capacity heatmaps.
- `IngestionPage.tsx`: Upload and log processing screen for uploading Archer CSV exports.
- `ReportsPage.tsx`: Generate executive PDF analysis, trends, and download operational spreadsheets.
- `TeamsPage.tsx`: Service Desk performance profiles, queues allocations, and resources productivity trends.
